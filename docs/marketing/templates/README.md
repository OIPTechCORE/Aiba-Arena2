---

## How to Customize

1. Open `.html` in a browser
2. Edit inline comments `<!-- CUSTOMIZE -->` or CSS `:root` variables
3. Resize: change `.banner` width/height in CSS
4. Export: DevTools → Screenshot, or Cmd/Ctrl+Shift+P → "Capture screenshot"
5. Or: Right-click element → Inspect → tweak → screenshot
