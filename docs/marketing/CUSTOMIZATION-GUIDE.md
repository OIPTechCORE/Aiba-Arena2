# AIBA Arena — Marketing Materials Customization Guide

**Scope:** How to customize all marketing assets. Replace placeholders, adjust copy, resize, and rebrand.

---

## Placeholders (Replace These)

| Placeholder | Replace With | Used In |
|-------------|--------------|---------|
| `{{APP_URL}}` | Your miniapp URL (e.g. https://t.me/YourBot/aiba) | All links |
| `{{REF_CODE}}` | Influencer/partner referral code | Influencer posts |
| `{{CALL_TO_ACTION}}` | Play Now / Join / Try Free | CTAs |
| `{{HASHTAG}}` | #AIBArena #PlayToEarn #TON | Social |
| `{{CAMPAIGN_NAME}}` | e.g. "Launch Week" | Campaign-specific |

## Brand Colors (CSS Variables — Match Globals)

| Name | Hex | Usage |
|------|-----|-------|
| Accent cyan | `#00d4ff` | Primary CTA, links |
| Accent gold | `#ffd700` | Premium, rewards |
| Accent magenta | `#ff00ff` | Boss, events |
| Accent green | `#00ff88` | Success, earn |

---

## Copy Variations by Channel

- **Twitter/X:** Short, punchy. Max 280 chars with link.
- **Telegram:** Casual, emoji-friendly. Include bot link.
- **Discord:** Community tone. Feature highlights.
- **YouTube:** Longer hooks. "In this video..."
