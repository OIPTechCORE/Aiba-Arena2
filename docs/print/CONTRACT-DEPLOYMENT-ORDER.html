Contract Deployment Order (TON / Tact)

This order matches current scripts and dependencies. Use testnet first.

1) Build

npm run build:contracts

2) Deploy order

  1. AibaJettonSupply (supply/minting)
  2. AibaToken (legacy token, if needed)
  3. ArenaVault
  4. ArenaRewardVault (requires AibaToken address)
  5. BrokerNFT
  6. BrokerMarketplaceEscrow
  7. AiAssetRegistry
  8. AiAssetMarketplaceEscrow
  9. AiAssetMarketplaceEscrowJetton
  10. MentorStakingVault

Scripts live in scripts/ and tact.config.json defines projects.

3) Backend env required for claims

Set these in backend:

  • AIBA_JETTON_MASTER
  • ORACLE_PRIVATE_KEY_HEX or ORACLE_SIGNER_URL
  • TON_PROVIDER_URL
  • TON_API_KEY

4) Validate

  • Run a small claim end‑to‑end with a test wallet
